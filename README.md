# 🚀 ITSkillRoute

## 📖 Жоба сипаттамасы
ITSkillRoute — IT саласын таңдауға көмектесетін веб-платформа.  
Пайдаланушы тест тапсырып, өзіне сәйкес бағытты анықтай алады және AI чат арқылы сұрақтарына жауап алады.

---

## 🛠 Қолданылған технологиялар (Tech Stack)
- HTML  
- CSS  
- JavaScript  
- Node.js (Express)  
- Google Gemini API  

---

## ⚙️ Жобаны іске қосу (Installation)

1. Репозиторийді жүктеу:
git clone https://github.com/your-username/itskillroute.git  

2. Папкаға кіру:
cd itskillroute  

3. Қажетті пакеттерді орнату:
npm install  

4. Серверді іске қосу:
node server.js
